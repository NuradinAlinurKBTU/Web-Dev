# Lab 7 - Task 2 (OOP in Python)

## Description
This project demonstrates Object-Oriented Programming concepts in Python.

## Structure
- models.py — contains classes
- main.py — runs the program

## OOP Concepts Used
- Base class: Animal
- Child classes: Dog, Cat
- Inheritance
- Method overriding (speak method)
- Polymorphism
- Encapsulation

## Example
Dog says "Woof", Cat says "Meow"
